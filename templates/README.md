# E-Commerce Website
Built a Flask-based E-Commerce Web Application with user authentication, admin product management, product search, shopping cart functionality, image support, and SQLite database integration.


## Features

- User Registration
- User Login
- User Logout
- Admin Login
- Add Products
- Edit products
- Delete Products
- Product images
- Product search
- Cart images
- Shopping Cart
- Remove Items from Cart
- Clear Cart
- Order history
- Total Bill Calculation
- SQLite Database

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
